聊聊毕业设计

毕业设计已经到了尾声，今天我们要聊的是我的毕业设计。
首先用到的技术有

  • 后端：
    • web
    • Spring MVC
    • web
    • Mybatis geneator
    • Mybatis pagehelper
    • Shiro
  • 前端：
    • EasyUI
    • Bootstrap
  • 数据库
    • postgreSql 9.3

用的是maven管理，其中用到了maven的插件：配置了jetty和tomcat。

毕设让我学到了很多，了解了很多，也让我了解到其实自己一个人做一个项目，不管他的大小如何，还是对一个人的能力有很多要求的，提醒自己需要不断的学习，因为知识确实是一个圈，懂的越多，发现自己不懂得越多。

毕设内容比较简单，是一个权限管理系统，可以动态的增删改查 user，role，task实现起来并不是很难，关键是权限的控制，其实很难的。最大的遗憾就是他的访问控制我还没有想到怎么解决，用的是shiro做权限控制，但是我对shiro又不是很熟悉，所以这个还没有实现，也就说一个人没有权限，他通过url也可以访问到他不能访问的东西。。。。想想都感觉致命。。。。
所以这是个失败的作品，他托管在https://git.oschina.net/shunming/UserPermission.git，有兴趣的可以下载下来看下，懂的人可以联系我，告诉我应该怎么解决。我的qq:1357197829。谢谢！！！:+1: :+1: :+1: :+1:
