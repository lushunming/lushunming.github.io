    zabbix 监控Oracle是使用一个插件orabbix,首先下载orabbix，并且解压到一个文件夹。首先打开conf中的conf.props,做如下修改

    • ZabbixServerList代表服务列表，一般我们就设置一个

    • ZabbixServer1.Address=服务的ip，前缀一定要和ZabbixServerList中的一样

    • ZabbixServer1.Port=服务的端口，前缀一定要和ZabbixServerList中的一样

    • oracle1.Url=数据库的连接信息

    • oracle1.User=数据库用户名

    • oracle1.Password=数据库密码

  3. 运行run.bat，启动orabbix，没有打印东西，说明启动成功。
